## 安装


### Win10

有手就能安装...

### Linux


添加 `x` 权限

```bash
chmod +x VMware-Workstation-Full-16.1.2-17966106.x86_64.bundle
```
使用 root 权限安装

```bash
sudo ./VMware-Workstation-Full-16.1.2-17966106.x86_64.bundle
```
