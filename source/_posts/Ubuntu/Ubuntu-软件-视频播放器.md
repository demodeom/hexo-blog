mpv 界面简洁、解码性能高，适合看电影；vlc 支持字幕、播放加速、视频列表、在线视频流等； 两个软件配合使用， 事半功倍

## mpv

[mpv 官网](https://mpv.io/)

```bash
sudo apt install mpv
```

## vlc

[vlc 官网](https://www.videolan.org/)

```bash
sudo apt install vlc
```
